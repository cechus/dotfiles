# 1. install zsh
```bash
sudo apt install zsh
```
# 2. install oh my zsh
```bash
sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```
# 3. run install.sh with sudo
```bash
sudo ./install.sh
```
**note:**
    Dar permisos de ejecucion `chmod +x install.sh`
